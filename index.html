A chrome extension I made with a group of 5 people. It allows users to upload their resumes, and when they are browsing LinkedIn job postings, they can use the extension to match uploaded resumes to the job posting to see an overall compatibility score and individual compatibility score for each resume so the user can easily determine which one is more suitable for the particular application.

The extension will also parse the posting and show the user keywords like required skills, job location, etc. In addition, it will present the user with a glassdoor review of the company.

React was used to build the extension's frontend, which is what I primarily worked on. StreamLit, FastAPI, LevelDB, and SpaCy were used for backend. Click here to check out the repository and here to watch a presentation video.

A smart trash grabber with an accompying mobile app I made with a team of 4 other people in 36 hours. Most parts on the grabber are 3D printed or laser cut. It features a servo-operated claw, allowing users to pick up trash with the click of a button. If the grabber detects trash being picked up with its depth sensor, it will beep and send GPS location data to the grabber's accompanying app.

The mobile app is a Pok√©mon-GO style Unity game. The main interface shows the user on a 3D map. People can report where trash is seen on the app, the locations of which will be sent to the backend made using Firebase, and trash hotspots will be calculated using the k means clustering algorithm and shown on the map. Whenever a user picks up trash, the app will receive GPS data from the grabber and display a flower for all other users on the map at the location indicated by the data. It will also increment user XP, incentivizing users to pick up more trash. The objective is to fill the map with as many flowers as possible to make it look like a blossoming garden.

I worked on the Unity app with a friend. Check out the Devpost and the respository.

A React web app I made with 4 other people. A user can create an account, choose an avatar, and interact with other users on a virtual platform. They can move their avatar around on a map, chat with other users in a global chat room, and create a little virtual booth with a zoom call link. If other users walk into the booth, they will be prompted with the link to join the call.

The frontend of the app is made using React and CSS, and the backend is powered by AWS with DynamoDB and Ably Realtime. I mostly worked on the frontend. Watch the video to see it in action and visit the respository.
