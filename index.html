A chrome extension I made with a group of 5 people. It allows users to upload their resumes, and when they are browsing LinkedIn job postings, they can use the extension to match uploaded resumes to the job posting to see an overall compatibility score and individual compatibility score for each resume so the user can easily determine which one is more suitable for the particular application.

The extension will also parse the posting and show the user keywords like required skills, job location, etc. In addition, it will present the user with a glassdoor review of the company.

React was used to build the extension's frontend, which is what I primarily worked on. StreamLit, FastAPI, LevelDB, and SpaCy were used for backend. Click here to check out the repository and here to watch a presentation video.

A Network Attached Storage I made using a Raspberry Pi 4 Model B, 4 hard drives, and a SATA hat I bought online. It's running Open Media Vault. I designed the case myself with Autodesk Inventor.

The NAS has a magnetically attached side panel, fan grill, and feet. I also made a external temperature-controlled fan using an arduino that can also be magnetically attached. Check out this video to see the fully assembled NAS and this video to see the fan module in detail.

A macro pad I made with a Raspberry Pi Pico. It's running MicroPython, features a a rotary encoder and a small OLED screen. It has several profiles that can be switched with a keypress, and the OLED screen will show a small bit image that match the profile.

The case and keycaps are designed in Autodesk Inventor and 3D printed. Watch this video to see the macro pad in action and this video to see the design.
